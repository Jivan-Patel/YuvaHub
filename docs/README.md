# YuvaHub Documentation Directory

Welcome to the YuvaHub documentation directory. Here you will find guides and reference documents regarding the platform requirements, development roadmap, local domains configuration, and server deployment methods.

## Document Catalog

### 1. [Product Requirements Document (PRD.md)](./PRD.md)
Contains the product specifications, core modules description, roadmap development plan (Phases 1-10), and the overall architecture flow of the application.

### 2. [Vercel Deployment Guide (DEPLOYMENT.md)](./DEPLOYMENT.md)
Step-by-step instructions on setting up and deploying the frontend client configuration on Vercel, including Google Gemini API key binding guides.

### 3. [Render Backend Deployment Guide (RENDER_DEPLOYMENT_GUIDE.md)](./RENDER_DEPLOYMENT_GUIDE.md)
Comprehensive migration guide to host the Express backend Web Service and the background cron-scheduler scraper on Render.com with MongoDB Atlas connectivity.

### 4. [Custom Domain & Firebase Setup Guide (DOMAIN_SETUP.md)](./DOMAIN_SETUP.md)
Instructions on mapping custom domains, configuring Firebase Authorized Domains to enable authentication correctly, and publishing DNS Service records (DNS-AID).
